# アンデッド従者システム仕様書

## 概要

アンデッド従者は、死体と魂を使った儀式で蘇らせる恒久的な従者である。
通常のペットとは別枠で管理され、死霊術向けの専用運用を前提としている。

最終確認: 2026-03-09

## 紹介ページ向け要約

- アンデッド従者はペット枠を使わない
- 通常ペットとは管理方式が異なる
- その代わり、通常ペットと同じ挙動をするわけではない

## アンデッド従者とペットの違い

| 項目 | アンデッド従者 | 通常ペット |
|---|---|---|
| 加入方法 | 儀式で蘇生する | 勧誘・加入イベント・通常の仲間化 |
| ペット枠消費 | しない | する |
| 所属の扱い | 従者として管理される | 仲間/ペットとして管理される |
| パーティ一覧ウィジェット表示 | 通常のパーティ一覧ウィジェットには出ない | 通常のパーティ一覧ウィジェットに出る |
| 仲間上限の表示 | 通常の仲間数/最大 ally 表示には含まれない | 通常の仲間数/最大 ally 表示に含まれる |
| 追従方式 | 従者専用の追従処理でついてくる | 通常のパーティ追従 |
| 通常の仲間加入 | 不可 | 可能 |
| 繋ぎ柱・通常騎乗導線 | 基本的に対象外 | 対象になる |
| 日常行動 | 本を読む、共有コンテナから物を取る、PCと一緒に釣るなどの一部「通常仲間向け」行動は行わない | こうした通常仲間向け行動を行うことがある |
| 専用蘇生 | 死亡後も従者として保持され、還生・再展開の対象になる | 通常の仲間/ペットの復帰ルールに従う |
| 召喚系能力 | 現在の仕様では使えない | 通常は使える |

## 補足

- アンデッド従者は「ペットの上位互換」ではなく、別ルールのユニット。
- 特に、ボスや特殊個体を従者化しても、元の挙動を完全再現するわけではない。
- 召喚系能力を持つ個体でも、従者化後はその能力を使えない場合がある。

## 紹介文の短文サンプル

> アンデッド従者は、通常のペットとは別枠で運用される恒久ユニットです。  
> ペット枠を消費しない代わりに、ロスター表示や一部の仲間向け挙動、召喚系能力などは通常ペットと同一ではありません。
