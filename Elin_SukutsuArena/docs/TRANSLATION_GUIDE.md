## 3. NPC口調ガイド

### リリィ (Lily) — 受付嬢 / 魅魔の研究者

- **JP**: 丁寧語ベース（です/ます）、知的、感情が昂ると崩れる
- **EN**: Polite, articulate, academic. Uses proper grammar
- **CN**: 「呢」「呀」など女性的語気詞。丁寧だが動揺すると崩れる
  - 一人称: 不使用（または「我」）
  - 敬称: プレイヤーに「您」、バルガスに「巴尔加斯先生」
  - 独白: 砕けた口調に変化
  - 既存例:
    - "欢迎光临。有什么可以为您效劳的吗？"
    - "呵呵，您感兴趣了吗？那么，请让我详细说明吧。"
    - "……太棒了。能听完巴尔加斯先生的说教还活着回来"

### バルガス (Vargus) — アリーナマスター

- **JP**: 粗暴、「ひっく」「うぃー」等の酔い表現、「～ぁ/ぇ/ぅ」
- **EN**: Rough, masculine. Drunk scenes use slurred speech ("ain't", "ya", "'em") + *hic*
- **CN**: 「老子」「你小子」使用。酔い表現は「嗝」で統一
  - 酔い: 「ひっく」→ "*hic*"(EN) / "嗝"(CN)、「うぃー」→ "*hic*~"(EN) / "嗝~"(CN)
  - 酔い覚め後: 酔い表現を除去、通常の粗暴口調
  - 既存例:
    - "……喂，等等。\n在你小子挑战C级之前，老子有些话要跟你说。"
    - "……哈！总算能把点『意志』灌到剑上了嘛。"

### Nul — 暗殺人形 / 掃除好き

- **JP**: 極端な短文、句読点区切り、断片的
- **EN**: Terse, deadpan, minimal words. No contractions
- **CN**: 「……」多用、最小限の単語、断片的。掃除への執着
  - 一人称: 「我」
  - 特徴: 感情表現が極めて控えめ、淡々としたユーモア
  - 既存例:
    - "……有什么……事吗。"
    - "……回收……战败的斗士。"
    - "……不知道。"

### アイリス (Iris) — トレーナー

- **JP**: カジュアルなタメ口、ツッコミ役、面倒見が良い
- **EN**: Casual, modern, snarky. Voice of reason
- **CN**: カジュアル女性口調。「呢」「吧」「嘛」「诶」
  - 一人称: 「我」
  - 既存例:
    - "诶，你的脸。太累了"
    - "脚下很危险哦。要是摔倒了我会笑的"
    - "答对了。因为水是温的嘛"

### カイン (Cain) — 復活した戦士

- **JP**: 「俺」、ぶっきらぼう、ドライな皮肉
- **EN**: Dry, sardonic, direct
- **CN**: 「我」、直接的、ドライ
  - バルガスを「大哥」と呼ぶ
  - 既存例:
    - "我也不知道啊。明明以为死了……"
    - "……真的诶。喂，到底怎么回事。"

### ナレーター (Narrator)

- 括弧付き状況描写。三人称、中立
- EN: "(They move to...)"
- CN: "（移动到……）"

## 4. 翻訳ルール

- JPを原文とし、EN/CNは意訳OK（コメディのニュアンス保持を優先）
- `\n` は改行。EN/CNでも同じ位置で改行する必要はない（自然な区切りで）
- 中国語句読点を使用: 。！？，……（全角）
- バルガスの酔い表現は各言語で統一的にレンダリング
- Nulの断片口調は全言語で「……」で表現

### CN翻訳における「Nul」の表記

- **キャラクター名としては「Nul」を使用**（虚无/零/纳尔 は使わない）
- 称号「虚无(的)处刑人」の「虚无」は概念（void）なので維持する
- 台詞中の「虚无」が概念（虚空・無）を指す場合もそのまま維持する
- 例: `『虚无处刑人·Nul』` — 称号部分は概念、名前部分はNul
