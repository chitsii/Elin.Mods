# アイリス（トレーナー）シナリオ

## 概要

アイリス（`sukutsu_trainer`）は3つのトレーニングシナリオを提供するトレーナーキャラクター。すべてのシナリオは**理不尽クイズ方式**で構成されており、試行錯誤を楽しみながら全問正解でバフを獲得できる。

**【登場人物】**
- **アイリス**：ノリ軽めのトレーナー。ギャルっぽい口調だが、時々不自然な知識を漏らす。高次元存在（Yith的存在）の可能性を示唆する台詞がある。
- **プレイヤー**：訓練を受ける側。クイズに正解するまでループし、間違えると軽い罰を受ける。

**アイリスの特徴：**
- ギャルっぽい口調（「～っしょ？」「ムカつく」）
- 時々不自然な知識を漏らす（閾値、アドレナリン、Yithの示唆など）
- 自分の体の反応に戸惑うことがある（「この体、勝手に落ちるんだよね」）
- 高次元存在の可能性（「円錐の器に宿り、あらゆる知識を収集し続ける」）

**理不尽クイズの哲学：**
- 正解はゲーム的な正解ではなく、キャラの内面や世界観に基づいている
- 「透明」「42」「無」「ヒラメ筋」「数えてない」「ぬるま湯」「わからない」「どっちでもない」など
- 試行錯誤を楽しみながら、アイリスの性格や価値観を理解していく構成
