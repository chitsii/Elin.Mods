# 第二部・第1話：蘇りの儀式

## 概要

バルガスとカインを蘇らせるためのイベント。第二部の最初に発生する必須イベント。
バルガスの生死状態によって導入部分が分岐する。

**必要素材（1人分）:**

**2人蘇らせるため、鶏は2羽必要**

## 【イベント効果】

### システム
- バルガスが復活（生存状態に）
- カインが新キャラとして加入
- ペットの鶏2羽が消費される
- 不老長寿のエリクシル×2、産卵薬×2が消費される
- フラグ：`BalgasRevived = true`
- フラグ：`CainRevived = true`

### 解放要素
- 第二部の他イベントが解放される
- 「大部屋の巻物」お披露目会が発生可能に
- カインが闘技場に常駐（バルガスの補佐役）

## 【シナリオ意図】

1. **第二部の幕開けとして** - バルガスとカインの復活で、第二部のトーンを確立
2. **新キャラ追加** - カインという濃いキャラで掛け合いのバリエーション増加
3. **Nulの深掘り** - 「守りたい人を覚えていない」という切ない設定
4. **コメディとシリアスのバランス** - 儀式自体は真剣だが、復活後は全力コメディ
5. **キャラクターの掛け合い** - 兄弟弟子のハモり、Nulの照れ
6. **伏線の回収と設置** - 請求書ネタ、カインの恩返し（後のイベントへ）
