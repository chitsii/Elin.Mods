# 第二部・第2話：大部屋の巻物お披露目会

## 概要

リリィが研究成果「大部屋の巻物」を発表するイベント。
実演の結果、闘技場が崩壊する大惨事に発展する。

**発生条件:**
- 蘇りの儀式完了後（バルガス＆カイン復活済み）

**イベント効果:**
- 「大部屋の巻物」がリリィの店に追加
- 闘技場が一時的に使用不可（再入場で自動復元）

## 【イベント効果】

### システム
- 「大部屋の巻物」がリリィの店に追加
- 闘技場が一時的に崩壊（Zone.forceRegenerate = true）
- ゾーンは再入場時に自動復元

### 実装メモ
```csharp
// 大惨事シーンで実行
Zone.forceRegenerate = true;
EClass.pc.MoveZone(exitZone); // 強制退場
```

## 【シナリオ意図】

1. **コメディ全開** - 第二部の明るいトーンを確立
2. **キャラクターの掛け合い** - バルガス＆カインの兄弟弟子コンビ、ヌルの体言止めツッコミ
3. **リリィの研究者キャラ** - 夢と現実のギャップで笑いを取る
4. **バルガスの脳筋ネタ** - 考えずに行動して大惨事
5. **請求書ネタの継続** - 第一部からの伏線回収＆第二部の定番ギャグに
6. **ゲームシステムとの連携** - 実際に闘技場が破壊される演出

## カインについて

カインは35年間死んでいたため、闘技場のことは何も知らない。
この話ではちょい出し程度の登場に留め、自虐ネタ（蘇ったばかり等）を担当。

カインが本格的に活躍するのは、闘技場に慣れた後の話から。
